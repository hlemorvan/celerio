= Celerio Code Generator
:toc:

NOTE: This project is a fork of https://github.com/jaxio/celerio, updated to run on modern Java and frameworks (Java 25, Spring Boot 4, Jakarta EE). +
Some documentation passages originate from https://www.jaxio.com/documentation/celerio/ and https://github.com/jaxio/jaxio.github.com. +
Other parts of the original documentation have not been carried over as they are no longer relevant to this version.

image:https://travis-ci.org/hlemorvan/celerio.svg[Build Status, link=https://travis-ci.org/hlemorvan/celerio]

Celerio is a code generator tool for data-driven applications. It can reverse a database schema and generate advanced CRUD-based applications.

* https://github.com/hlemorvan/celerio/blob/master/celerio-documentation/src/main/asciidoc/master.adoc[Celerio Documentation]

== How to use it?

Make sure you have Maven 3.6+ and Java 25 installed, then execute:

----
mvn com.jaxio.celerio:bootstrap-maven-plugin:bootstrap
----
